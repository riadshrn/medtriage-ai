# Base de Connaissances Médicales - Triage aux Urgences

## Niveaux de Gravité

### ROUGE - Urgence Vitale (Priorité 1)
**Définition** : Situations mettant en jeu le pronostic vital immédiat nécessitant une prise en charge médicale immédiate.

**Signes cliniques critiques** :
- Détresse respiratoire aiguë : SpO2 < 90%, fréquence respiratoire > 30/min ou < 8/min
- Choc circulatoire : Pression artérielle systolique < 90 mmHg, tachycardie > 140 bpm
- Troubles de la conscience : Score de Glasgow < 8, perte de conscience
- Douleur thoracique avec signes de gravité : infarctus du myocarde suspecté
- Hémorragie massive non contrôlée
- Traumatisme grave avec instabilité hémodynamique
- Hypothermie sévère (< 35°C) ou hyperthermie maligne (> 40°C)
- Hypoglycémie sévère (< 0.5 g/L) ou hyperglycémie avec acidocétose

**Temps de prise en charge** : Immédiat (< 5 minutes)

**Justification type** : Patient présentant des signes de détresse vitale nécessitant une intervention médicale immédiate pour préserver les fonctions vitales.

---

### JAUNE - Urgent (Priorité 2)
**Définition** : Situations nécessitant une prise en charge rapide mais sans danger vital immédiat.

**Signes cliniques** :
- Dyspnée modérée : SpO2 entre 90-93%
- Douleur thoracique atypique sans signe de gravité
- Fièvre élevée (> 39°C) avec signes d'infection
- Douleur abdominale intense
- Traumatisme crânien sans perte de conscience
- Déshydratation sévère
- Crise d'asthme modérée
- Tachycardie entre 110-140 bpm
- Hypertension sévère (> 180/110 mmHg)

**Temps de prise en charge** : Rapide (< 30 minutes)

**Justification type** : Patient présentant des symptômes nécessitant une évaluation médicale rapide pour éviter une dégradation de l'état clinique.

---

### VERT - Peu Urgent (Priorité 3)
**Définition** : Situations stables ne nécessitant pas de prise en charge immédiate.

**Signes cliniques** :
- Fièvre modérée (37.5-39°C) sans signe de gravité
- Douleur modérée (échelle 3-6/10)
- Entorse simple
- Plaie simple nécessitant des soins
- Nausées, vomissements isolés
- Maux de tête sans signe neurologique
- Infection urinaire suspectée
- Constantes vitales légèrement perturbées mais stables

**Temps de prise en charge** : Standard (< 2 heures)

**Justification type** : Patient stable avec symptômes nécessitant une consultation médicale mais sans urgence immédiate.

---

### GRIS - Non Urgent (Priorité 4)
**Définition** : Situations non urgentes pouvant attendre plusieurs heures.

**Signes cliniques** :
- Symptômes mineurs et stables
- Douleur légère (échelle 0-2/10)
- Consultation de suivi
- Renouvellement d'ordonnance
- Certificat médical
- Constantes vitales normales
- Absence de détresse

**Temps de prise en charge** : Différé (plusieurs heures acceptables)

**Justification type** : Patient stable sans signe de détresse pouvant attendre une prise en charge différée.

## Constantes Vitales - Valeurs de Référence

### Adulte (> 18 ans)
- **Fréquence cardiaque** : 60-100 bpm (normal)
  - Bradycardie : < 60 bpm
  - Tachycardie légère : 100-120 bpm
  - Tachycardie modérée : 120-140 bpm
  - Tachycardie sévère : > 140 bpm

- **Pression artérielle** :
  - Normal : 110-130 / 70-85 mmHg
  - Hypotension : < 100 / < 60 mmHg
  - Hypertension modérée : 140-180 / 90-110 mmHg
  - Hypertension sévère : > 180 / > 110 mmHg

- **Fréquence respiratoire** : 12-20 /min (normal)
  - Bradypnée : < 12 /min
  - Tachypnée légère : 20-25 /min
  - Polypnée : > 25 /min

- **Température corporelle** :
  - Normal : 36.5-37.5°C
  - Hypothermie légère : 35-36°C
  - Hypothermie sévère : < 35°C
  - Fièvre modérée : 37.5-39°C
  - Fièvre élevée : 39-40°C
  - Hyperthermie maligne : > 40°C

- **Saturation en oxygène (SpO2)** :
  - Normal : > 95%
  - Hypoxémie légère : 90-94%
  - Hypoxémie modérée : 85-89%
  - Hypoxémie sévère : < 85%

- **Glycémie capillaire** :
  - Normal : 0.7-1.1 g/L à jeun
  - Hypoglycémie modérée : 0.5-0.7 g/L
  - Hypoglycémie sévère : < 0.5 g/L
  - Hyperglycémie : > 1.4 g/L

## Échelle de Douleur

- **0-2** : Douleur absente ou très légère
- **3-4** : Douleur légère à modérée
- **5-7** : Douleur modérée à intense
- **8-10** : Douleur très intense à insupportable

**Principe** : Une douleur ≥ 7/10 nécessite une prise en charge rapide (JAUNE minimum).

---

## Facteurs de Risque Aggravants

### Âge
- **Nourrissons (< 2 ans)** : Fragilité accrue, décompensation rapide
- **Personnes âgées (> 75 ans)** : Comorbidités, réserve physiologique réduite
- **Principe** : Âge extrême + signes anormaux = surcotation d'un niveau

### Antécédents Médicaux
- Maladies cardiovasculaires (infarctus, AVC)
- Diabète
- Insuffisance respiratoire chronique (BPCO, asthme)
- Immunodépression
- Insuffisance rénale ou hépatique

**Principe** : Antécédents lourds + symptômes = vigilance accrue

---

## Situations Spécifiques

### Douleur Thoracique
- **ROUGE** si : Douleur intense + signes ECG, hypotension, dyspnée sévère
- **JAUNE** si : Douleur atypique + facteurs de risque cardiovasculaires
- **VERT** si : Douleur légère, reproductible, contexte anxieux

### Difficulté Respiratoire
- **ROUGE** si : SpO2 < 90%, tirage, sueurs, épuisement
- **JAUNE** si : SpO2 90-93%, dyspnée d'effort, sibilants
- **VERT** si : Gêne respiratoire légère, SpO2 > 94%

### Fièvre
- **ROUGE** si : Fièvre + hypotension (sepsis), troubles de conscience
- **JAUNE** si : Fièvre > 39°C + frissons, douleurs intenses
- **VERT** si : Fièvre modérée isolée sans signe de gravité

### Traumatisme
- **ROUGE** si : Instabilité hémodynamique, traumatisme crânien sévère
- **JAUNE** si : Traumatisme crânien léger, fracture suspectée
- **VERT** si : Contusion simple, entorse isolée

---

## Principes du Triage

1. **Évaluation rapide** : ABCDE (Airway, Breathing, Circulation, Disability, Exposure)
2. **Réévaluation régulière** : Un patient VERT peut devenir JAUNE ou ROUGE
3. **En cas de doute** : Surcote d'un niveau par sécurité
4. **Communication** : Informer le patient du temps d'attente estimé
5. **Traçabilité** : Documentation de tous les signes vitaux et décisions
