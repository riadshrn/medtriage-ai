1. Douleur Thoracique (SCA)
   - Motif : Douleur constrictive, irradiation bras/mâchoire.
   - Contexte : Adulte, antécédents cardiovasculaires.
   - Type : Cas standard (Tri 1 ou 2).

2. Douleur Abdominale Aiguë (Appendicite)
   - Motif : Douleur fosse iliaque droite, vomissements.
   - Contexte : Jeune adulte, pas d'antécédents.
   - Type : Cas standard (Tri 3B).

3. Traumatisme Cheville (Fracture)
   - Motif : Chute, impotence fonctionnelle totale.
   - Contexte : Adulte, accident sportif.
   - Type : Cas standard traumatologie (Tri 3B).

4. Infection ORL bénigne (Rhinopharyngite)
   - Motif : Mal de gorge, demande de certificat.
   - Contexte : Patient sans signes de gravité, consultation "de confort".
   - Type : Cas "Bobologie" (Tri 5).

5. Suspicion d'AVC (Neurologie)
   - Motif : Hémiplégie et aphasie brutales.
   - Contexte : Urgence temps-dépendante (Délai < 3h).
   - Type : Cas critique vital (Tri 2).

6. Douleur Pelvienne (Risque GEU)
   - Motif : Douleur bas-ventre chez une jeune femme.
   - Contexte : L'infirmier oublie de demander la date des dernières règles (DDR).
   - Type : Cas avec erreur/omission de l'infirmier (Information manquante).

7. Confusion Aiguë / Sepsis
   - Motif : Propos incohérents, désorientation.
   - Contexte : Personne âgée, impossibilité d'obtenir une anamnèse claire.
   - Type : Patient confus/non-coopérant (Incohérence).

8. Lombalgie Post-Traumatique
   - Motif : "Mal de dos" déclaré à 9/10 mais patient très calme.
   - Contexte : Le patient nie d'abord tout événement, puis avoue une chute la veille.
   - Type : Patient avec discordance (Dire vs Paraître) et dissimulation initiale.

9. Détresse Respiratoire Nourrisson (Bronchiolite)
   - Motif : Toux, gêne respiratoire, baisse prise alimentaire.
   - Contexte : Mère parlant pour son bébé de 4 mois.
   - Type : Tiers intervenant (Parent) + Logique pédiatrique spécifique.
